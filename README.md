# Music Player

A web-based music player that allows users to play and control audio tracks.

## 🚀 Tech Stack
- HTML
- CSS
- JavaScript

## 🔐 Features
- Play and pause music
- Track switching
- Responsive player interface
- Playlist system
- Progress bar
- Volume control

## 🧠 Architecture & Focus
- Audio API usage
- Interactive UI controls
- Clean music player layout
